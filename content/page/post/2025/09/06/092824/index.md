はてなブログの中のいくつかの記事を、より特化した新しいブログへ移転することにした。

元記事へのアクセスも一定数あるため、しばらくは移転元の記事から移転先の記事へ自動で案内（リダイレクト）するよう設定しておきたい。

はてなブログでは301リダイレクトは設定できないようだったが、以下の手順であれば問題なく移行できたので、備忘録としてやり方を残しておきたい。

今後は、折を見てGoogleサーチコンソールからの削除なども進めていきたい。

<h3 id="フッターによるjQueryの読み込み">フッターによるjQueryの読み込み</h3>

はてなブログの「管理画面」から「デザイン→カスタマイズ→フッタ」を開き、「フッタ」に下記のスクリプトを追記する。


```
<!--jQuery-->
<script type="text/javascript" src="http://code.jquery.com/jquery-1.9.1.min.js"></script>
```



<h3 id="移転先記事の作成">移転先記事の作成</h3>

移転先のURLに、移転させたい内容の記事を作成する。

基本的には移転元の内容をコピペすればよいが、移転にあたってはリンク切れなどないように注意したい。

<h3 id="移転元の記事をリダイレクト設定するよう編集">移転元の記事をリダイレクト設定するよう編集</h3>

移転元となるはてなブログ記事に、下記のスクリプトをコピペする。

※2か所ある「★移転先のURL★」は、移転対象となる記事のURLに差し替えること


```html
この記事は移転しました。約2秒後に新記事へ移動します。自動で移動しない場合は<a href='★移転先のURL★'>こちら</a>をクリックしてください。

<script type='text/javascript'>
// 記事ページのリダイレクト
if (/entry/.test(window.location.href)) {
  setTimeout(function() {
    location.href = '★移転先のURL★';
  }, 2000); // 2秒後にリダイレクト
}
</script>
<hr>

```



<h3 id="動作の確認">動作の確認</h3>

移転元の記事を開いて、自動的に移転先に飛ぶか確認する。

念のため、リンクをクリックしても移動できるかチェックしておく。

<h3 id="最後に">最後に</h3>

以上の方法で、はてなブログでも簡易的なリダイレクトが可能になる。

恒久的な SEO 対応には301リダイレクトが望ましいが、はてなブログの仕様上難しいため、当面はこの方法で対応するのが現実的と考えている。
