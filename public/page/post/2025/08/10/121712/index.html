【はてなブログ】無料でトップページを設置

August 10, 2025 · 1 min · ip-note

はてなブログは基本トップページに記事全文が表示される仕様となっている。

有料版（Pro）であれば、記事一覧の表示を選べるが、無料版の場合は一覧形式で表示できない。

そして、どうやらトップページをアーカイブにリダイレクトする方法は利用規約違反らしい。

そこで、今回は固定ページを作り、そこで一覧表示をはじめ、自分の思い通りのデザインにできるようなカスタマイズ方法を紹介したい。

なお、この方法はプライバシーポリシーといったページを設ける際にも応用できる部分があるかと思う（この場合は、投稿日時を未来にしたり「トップページの先頭に表示」といった対応は不要かと思う）。

  • [その他非表示設定](#その他非表示設定) * [アーカイブ（記事一覧）から非表示にする](#アーカイブ記事一覧から非表示にする) * [注目記事、最新記事一覧から非表示にする](#注目記事最新記事一覧から非表示にする) 
    </li>

    記事を書く#

    通常の記事と同じようにトップページ用の記事を書く。

    ここで記事を自分の思うようにデザインし、また必要に応じて自分で選んだ記事を一覧表示させる。

    例えば、当ブログでは以下のような構成としている。

    グルメ漫画を味わい尽くす

    トップページ用のコードを入力#

    トップページらしくなるよう、記事の中に不要な要素を画面から消す以下のコードを入力する。

    <style>
  /* サイドバー、ヘッダー/フッター（タイトル、日付など）、ページャーを非表示 */
  #box2,
  .entry-footer,
  .entry-header,
  .pager{
    display: none;
  }

  /* サイドバーを非表示とした分、メインコンテンツを横いっぱいに広げる */
  #main {
    width: 100% !important;
    max-width: 100% !important;
    float: none !important;
  }
</style>

    トップページ記事の投稿#

    準備ができたら、以下の手順で記事を投稿・設定し、トップページ記事のみが表示されるようにする。

    • 完成した「トップページ記事」の投稿日時を「9999年」などの遠い未来の数字にする（トップページ記事が最新記事として最初に表示されるようになる）
    • 他の通常記事と同じように一般公開する
    • 「記事の管理」の「記事タイトル」一覧で、「トップページ記事」のタイトル右にある「･･･」から「トップページの先頭に表示」を選択する
    • 「設定」→「 詳細設定」 → 「トップページの記事数（PC版）」で、記事数を１記事に設定

    その他非表示設定#

    このままだと、アーカイブページや、サイドバーのモジュールに入れた注目記事、最新記事一覧にトップページ記事が出てきてしまう。

    そこで、これら一覧からトップページ記事が表示されないようにCSSコードを入力する。

    アーカイブ（記事一覧）から非表示にする#

    • 記事一覧ページ（ブログドメインの末尾に「/archive」を付したページ）を開く
    • 一覧から、トップページ記事のタイトル上で右クリックし、「検証」を選択
    • 各記事を構成する以下のようなセクションタグから、data-uuid=“19桁のID番号"のID番号部分をコピー
    <section class='archive-entry test-archive-entry autopagerize_page_element' data-uuid='19桁のID番号（トップページ記事のID番号）'>
    • 「デザインCSS」（「デザイン」→「カスタマイズ」→「デザインCSS」）を開き、以下のコード（「★トップページ記事のID番号★」の部分は、上でコピーしたIDに入れ替える）を貼り付ける
    .archive-entry[data-uuid='★トップページ記事のID番号★'] {
    display: none;
}

    注目記事、最新記事一覧から非表示にする#

    • サイドバーの注目記事にあるトップページ記事のタイトル上で右クリックし、「検証」を選択
    • 以下のようなHTMLが表示されるので、「★トップページ記事のURL★」の部分をコピーしておく
    <ul class='hatena-module-list'>
  <li>
    <a href='★トップページ記事のURL★'>トップページ記事のタイトル</a>
  </li>
</ul>
    • 「デザインCSS」を開き、以下のコード（「★トップページ記事のURL★」部分は、上でコピーしたURLに入れ替える）を貼り付ける
    /* 注目記事の中で「トップページ記事」のリンクを含む<li>全体を非表示にする */
.hatena-module-body li:has(a[href='★トップページ用記事のURL★']) {
  display: none !important;
}

    プライバシーポリシーのページなど、トップページと同様に非表示としたいページがあれば、そのページについても「その他非表示設定」の作業を行えば良い。
